TEST How to Install VS Code's Live Server Extension

Install| Use |Video

Installation Instructions

  1. In Visiual Studio Code , click the Extensions icon in the left colum toolbar.
  2. In the search bar that appears, type Live Server.
  3. Select the Live Server Extension. It should be the top choice.
  4. In the page for Live Server that appears, click the green install button.

Usage Instructions

  • visual Studio Code must be open to a forlder to use Live Server.
  • In VS Code's Explorer, right click (windows) or contro+click (macOS) on an HTML file. From the context menu that appears, select Open with Live Server.
